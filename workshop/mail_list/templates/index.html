CODE42DAY

What is CODE42DAY ??

CODE42DAY is a development side coding challenge. It is platform and programming language independent. Yes, you can code in any language you are confortable and work for any platform(Windows/Linux/Android /IOS). It will begin around January and will test you in different aspect of it.

How to begin ??

You will have to submit your ideas till first week of January. You can discuss your ideas with us to know feasibility and how to work on it. We will guide you throughout the contest.
If you are not comfortable in programming yet, work on it in Dushehrraa and Winter vacations. Read about the technology taught in workshop and do work on further reading section provided in PPTs.
Note: The concept taught in workshop does not depend on programming language, if you have no knowledge of programming you can still excel in the workshop.
If you have experience in programming sharp your skills by creating simple code script which solves your daily problems you feel like could be coded. Think of an idea to work on for the event and discuss with us and start coding.
If you are unable to get any new idea, you can take on an existing open-source project and add a new desired functionality.
Feel free to contact us for any related problems you face through e-mail, whatsapp or facebook.

What will be tested ??

  • It is a team contest, hence we will check team collaboration using 'Git'.
  • Programming skills.
  • Use of database or caching.
  • Use of APIs is a bonus.
  • Number of functionality available
A separate prize for best fist year team* :)

There will be no marking on Graphical User Interface(GUI)
